Astro 正式加入 Cloudflare 家族

開源框架獲得企業級後盾，開啟全新篇章

這是前端界的重磅消息！Astro Technology Company 宣布被 Cloudflare 收購，所有全職員工現已成為 Cloudflare 團隊成員。對於使用 Astro 的開發者來說，這是個好消息——Cloudflare 的資源將確保框架的長期穩定發展。

語言演進

TypeScript 進入 6.0 維護模式

Go 語言重寫的 TypeScript 7.0 即將登場

2026年1月15日 · Microsoft TypeScript 團隊

Microsoft 宣布 TypeScript 6.0 將是最後一個基於 JavaScript 程式碼庫的版本，團隊正全力投入 Go 語言重寫的 TypeScript 7.0。由於「Go 語言重寫代表了開發的不可避免斷層」，大部分目前針對 JavaScript 程式碼庫開啟的 Pull Request 將不會被合併。

6.0 僅接受以下類型的變更：與 6.0 重大變更相關的必要工作、關鍵性的版本回歸修復、6.0 與 7.0 行為對齊的變更，以及可在兩個版本重現的崩潰修復。TypeScript 團隊表示：「感謝多年來對 TypeScript 的精彩貢獻。」

瀏覽器 API

URL Pattern API 正式可用

path-to-regexp 即將成為時代的眼淚？

2026年1月 · Baseline 2025

MDN 宣布 URL Pattern API 已成為 Baseline 2025，可在所有主流瀏覽器中使用。支援命名群組、正則約束、可選項、萬用字元等模式類型。

const pattern = new URLPattern({ pathname: "/books/:id" }); pattern.test("https://example.com/books/123"); // true const result = pattern.exec("https://example.com/books/123"); console.log(result.pathname.groups); // { id: '123' }
